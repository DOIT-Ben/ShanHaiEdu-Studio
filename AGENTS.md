# 项目级 Codex 工作规则

> 本文件约束当前项目 `shanhai媒体工作台`。若与用户当前明确指令冲突，以用户当前指令为准；若与平台、安全或工具权限限制冲突，以平台和工具限制为准。

## 项目概况

- 项目名称：shanhai媒体工作台
- 项目类型：独立前端原型 / 可运行 Web 工作台
- 技术栈：Next.js、React、TypeScript、Tailwind CSS、Radix UI、lucide-react
- 主要入口：`src\app\page.tsx`
- 主要输出物：本地可预览的线性 AI 备课媒体工作台

## 工作边界

- 本项目是独立桌面目录，不得修改原 ShanHaiEdu 仓库。
- 所有演示数据必须明确为演示态，不得伪装成真实生成结果。
- 用户可见界面不得出现工程调试词、后端字段名、命令行、接口或路径。
- 优先复用成熟 UI primitives，不重做一套 UI kit。

## 运行与验证

- 安装依赖：`npm install`
- 开发启动：`npm run dev`
- 构建验证：`npm run build`

## 视觉规则

- 工作台不是营销页，不做 hero、大渐变背景或炫技滚动动画。
- 字体最大不超过 35px。
- 颜色只按语义使用：主色深青灰、焦点古铜金、成功、警告、信息、危险。
- 圆角控制在 8px 左右。
- 桌面端和窄屏都要做浏览器检查。
