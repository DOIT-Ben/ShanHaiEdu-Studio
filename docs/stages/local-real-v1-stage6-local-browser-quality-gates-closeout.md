## 1. 范围

本切片关闭 M67 反馈中心桌面与窄屏浏览器门禁，并刷新当前 V1 工作树的全量测试、生产构建和 SQLite 幂等初始化证据。本文件不代表三套真实课程已经完成，也不代表目标服务器或真实教师验收已经通过。

## 2. 修复与契约

- 恢复认证后欢迎页的全局反馈入口和窄屏账户菜单。
- 恢复反馈影响程度、成功回执和消息评价后的引导反馈。
- 消息评价先持久化；取消评价不打开反馈，新增或切换评价才打开对应反馈。
- CSV 导出对每个逻辑行的公式前缀执行文本化处理。
- CSV 导出显式引用包含 CR/LF 的字段，避免 UI 多行反馈被拆成多个记录。
- 增加超过 100 条、危险值位于后续分页的回归覆盖。
