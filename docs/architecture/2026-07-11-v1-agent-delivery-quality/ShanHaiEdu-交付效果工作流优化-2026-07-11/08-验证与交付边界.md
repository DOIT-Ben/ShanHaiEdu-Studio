# 验证与交付边界

## 已完成验证

- Contract Registry：22 个 Contract，ID 唯一；`recommendedNext` 仅做建议引用，引用目标完整，不作为全局 DAG 验收。
- Contract enforcement：69 条禁止规则为 MUST；57 条质量标准为 MUST；36 条质量标准为 SHOULD。
- JSON Schema：DeliveryStrategy、PPT Director 整包、PageSpec、ShotSpec、ReviewFinding、AgentDecisionEnvelope、ToolObservationV2 共 8 组示例离线验证通过。
- ReAct 语义：总体设计、总控 Prompt、Agent 决策投影和 Tool Observation 已对齐；最新独立审查见 `review/independent-agent-react-adaptation-review.md`。
- 官方来源连通性：OpenAI Agents/Runner、Anthropic、Google ADK、LangGraph、Temporal 和 ReAct 关键页面在 2026-07-11 实测返回 HTTP 200。
- 图片实验：两张 1672×941 真实图片，Prompt、Provider profile、模型和 SHA256 已记录。
- 视频实验：6.041667 秒 MP4 完整解码通过；blackdetect 与 freezedetect 无命中。
- 视频失败证据：实际 752×416、额外 MJPEG attached-picture 流、模糊展签、综合响度 -32.3 LUFS。
- 独立智能体：旧版内容流程审查与新版 ReAct 适配审查分开保留；新版审查覆盖 Prompt/Schema、动态计划、Tool 分层、框架取舍和代码插入点。
- 最终目录：由 `generate-delivery-manifest.ps1` 对实际文件生成 SHA256 清单。
