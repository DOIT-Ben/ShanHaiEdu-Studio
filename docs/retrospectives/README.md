# ShanHaiEdu 复盘总则

本目录记录已发生的交付偏差、根因、可复用规则和未闭环行动。事实、推断和未验证项必须分开；写入文档不等于问题已经修复。

## 跨场景规则

1. 技术链路通过不等于用户交付通过；内容效果必须由对应领域 Rubric 独立验收。
2. 导入视频必须先作为脱离教材页面、PPT结构、教师讲解和课堂角色仍成立的独立短片，再用唯一课程锚点回接学习任务。
3. 未经 Concept Selection Gate，不得从教案、教材情境或PPT叙事直接生成 Beat Sheet 和 ShotSpec。
4. 模拟集成批准只能验证生产链，不得代替创意方向批准或真实教师签收。
5. 验收包任一核心产物不合格，整包不得标记完整交付。
6. 产品外部由 Codex 人工编排得到的优质产物，只能证明方法和 Provider 可行，不能证明产品内部 Agent 已具备同等规划、工具调用、打断恢复和质量决策能力。
