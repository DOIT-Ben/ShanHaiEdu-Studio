# Local Real MVP M72 测试计划

更新时间：2026-07-11

## 1. 反馈体验

- 打开反馈弹窗后依次切换七类反馈，验证弹窗尺寸、页脚和滚动位置稳定。
- 验证快速补充内容替换但区域始终占位。
- 桌面与 390px 检查默认、选中、提交中、成功和失败状态。
- 点赞、点踩、切换、取消后出现可访问提示，刷新后状态恢复。

## 2. 对话与工具调用

- 有活动低副作用计划时输入“确认需求并生成大纲”，能够执行。
- 无活动计划时直接输入“只做视频脚本”，进入对应能力，不要求先完成 PPT。
- 缺少硬前置时只说明最小缺口并创建下一动作，不返回通用“没有有效确认”。
- 真实 PPTX、图片、视频和最终包调用前仍需已披露动作的明确确认。
- 改道后旧 actionId 失效；policy 拒绝不计入工具失败重试预算。

## 3. 多用户与数据归属

- 指定账号可通过密码登录，密码仅以 salted scrypt 哈希保存。
- 迁移前后项目、消息、产物数量一致；项目 owner/membership 指向指定账号。
- 第二个教师账号无法列出、读取或下载第一个账号未共享的数据。
- 管理员邀请、停用、重置凭据继续要求认证和 CSRF，响应不回显密码。

## 4. 工程门禁

- `npx tsc --noEmit --incremental false`
- 相关 Node/Vitest 定向测试
- `npm test`
- `npm run build`
- `git diff --check`
- 真实浏览器桌面与 390px 验收
