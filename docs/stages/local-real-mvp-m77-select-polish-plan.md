# Local Real MVP M77 下拉框统一优化计划

更新时间：2026-07-11

## 目标与范围

- 将成员权限和账号角色的原生下拉框迁移到项目统一 Select。
- 统一触发器、弹层、选中项、悬浮项和键盘焦点视觉。
- 不修改权限枚举、成员管理 API、用户管理 API 或保存行为。

## 风险与回退

- 风险：迁移时权限值类型或保存回调改变。通过保留原枚举值并执行成员管理回归测试控制。
- 回退：可独立恢复三个选择器和 `select.tsx` 样式，不涉及数据迁移。

## 验收

- 项目源码无原生 `<select>`。
- 可查看、可编辑、教师、管理员选项与回调保持一致。
- TypeScript、定向测试、全量测试和生产构建通过。
