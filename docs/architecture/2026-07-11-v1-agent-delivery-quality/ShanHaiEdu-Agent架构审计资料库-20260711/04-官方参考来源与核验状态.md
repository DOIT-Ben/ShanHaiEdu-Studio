## 7. 尚未被来源证明的事项

以下必须通过 Spike，当前不能写成既成事实：

- LangGraph 在当前本地 SQLite、Next.js 运行模式下的实际恢复可靠性。
- 任一框架与现有 OpenAI-compatible Provider 的完整兼容性。
- 真实 PPT/视频长任务的成本、延迟和内存占用。
- Provider 已提交后进程崩溃时的 exactly-once 效果。
- 自然语言回跳时 Artifact stale 传播是否符合教师预期。
- UI 断线重连后 Tool、审批和子图事件能否无重复恢复。
